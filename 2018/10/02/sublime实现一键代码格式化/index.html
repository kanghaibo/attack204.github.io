用Astyle实现sublime代码一键格式化

效果预览

实现

首先下载插件SublimeAstyleFormatter

方法：ctrl + shift + P后输入install Package。

等待一段时间后输入SublimeAstyleFormatter即可自动下载

使用方法

按下ctrl + Alt + F是默认格式化整个文件

按下ctrl + K + F即可格式化选中区域

该插件中有许多不同的风格可以选择

我比较习惯google代码风格，可以这样设置

然后把第$28$行改为

"style": "google",

完成！
